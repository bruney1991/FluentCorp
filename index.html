Fluent Corp

Aplicativo de aprendizado de idiomas desenvolvido com Flutter e Node.js

Acessar Repositório no GitHub

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma plataforma interativa, simples e acessível.

O sistema utiliza uma arquitetura cliente-servidor composta por aplicativo mobile, API backend e banco de dados relacional.

Tecnologias Utilizadas

Flutter

Framework utilizado para desenvolvimento da interface mobile.

Node.js

Responsável pelo backend e criação da API REST.

Sequelize

ORM utilizado para comunicação com o banco de dados.

MySQL

Banco de dados utilizado para armazenamento das informações.

Funcionalidades

Cadastro e autenticação de usuários
Seleção de idioma para aprendizado
Definição de nível de conhecimento
Flashcards para memorização
Treinamento de conversação
Conteúdos em áudio (músicas e podcasts)
Perfil do usuário e progresso
