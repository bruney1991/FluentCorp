Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma interface simples e interativa.

O sistema possui uma API desenvolvida em Node.js conectada ao banco de dados MySQL e uma interface mobile desenvolvida em Flutter.

Tecnologias Utilizadas

📱 Flutter

Interface mobile do aplicativo.

⚙️ Node.js

Backend responsável pela API.

🧠 Sequelize

ORM utilizado para integração com o banco.

🗄️ MySQL

Banco de dados do sistema.
