Fluent Corp

Aplicativo de aprendizado de idiomas desenvolvido com Flutter e Node.js

Acessar repositório

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma interface simples, interativa e personalizada.

O sistema utiliza uma API em Node.js conectada a um banco MySQL e uma interface mobile desenvolvida em Flutter.

Tecnologias Utilizadas

📱 Flutter

Framework utilizado para desenvolvimento da interface mobile.

⚙️ Node.js

Backend responsável pela API e comunicação com o banco de dados.

🧠 Sequelize

ORM utilizado para integração entre o Node.js e o banco MySQL.

🗄️ MySQL

Banco de dados responsável pelo armazenamento das informações.

Arquitetura do Sistema

O sistema segue uma arquitetura cliente-servidor.

Flutter → API Node.js → Banco de Dados MySQL

Funcionalidades

👤 Cadastro de usuários

Criação de conta para acesso ao aplicativo.

🔐 Login seguro

Autenticação de usuários na plataforma.

🌍 Seleção de idioma

Escolha do idioma para aprendizado.

📊 Nível de conhecimento

Definição do nível do usuário no idioma escolhido.

🗓️ Planejamento de estudos

Usuário define disponibilidade semanal para organizar os estudos.

🧠 Flashcards

Sistema de memorização de vocabulário.

💬 Treino de conversação

Prática de diálogo para melhorar fluência.

🎧 Conteúdo em áudio

Músicas e podcasts para prática de escuta.

📈 Perfil e progresso

Acompanhamento da evolução do usuário.
