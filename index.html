Acessar Repositório

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma experiência interativa e personalizada.

O sistema utiliza Flutter no aplicativo mobile e uma API construída em Node.js com Sequelize para comunicação com banco de dados MySQL.

Tecnologias Utilizadas

📱 Flutter

Interface mobile moderna e responsiva.

⚙️ Node.js

Backend responsável pelas regras de negócio.

🗄️ Sequelize

ORM utilizado para comunicação com o banco.

💾 MySQL

Armazenamento e gerenciamento de dados.

Arquitetura do Sistema

Fluxo de comunicação entre aplicativo, API e banco de dados.
