Sobre o Projeto

O Fluent Corp é um projeto acadêmico focado em facilitar o aprendizado de novos idiomas com uma interface moderna e interativa.

Utilizamos Node.js com Sequelize para o backend e Flutter para uma experiência mobile fluida.

Tecnologias

📱 Flutter

Interface Mobile Híbrida.

⚙️ Node.js

API REST e lógica de negócio.

🗄️ MySQL

Banco de dados relacional.

🔗 Sequelize

ORM para persistência de dados.
