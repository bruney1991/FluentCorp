Sobre o Projeto

O Fluent Corp é um aplicativo acadêmico que auxilia no aprendizado de idiomas com interface simples, moderna e interativa.

O sistema usa Flutter no front, Node.js + Sequelize no backend, e MySQL como banco de dados.
