Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma interface simples, moderna e interativa.

O sistema utiliza uma API desenvolvida em Node.js com Sequelize para comunicação com banco de dados MySQL e uma aplicação mobile desenvolvida em Flutter.

Tecnologias Utilizadas

📱

Flutter

Framework utilizado para desenvolvimento da interface mobile.

⚙️

Node.js

Backend responsável pela API e comunicação com o banco de dados.

🗄️

Sequelize

ORM utilizado para interação com o banco de dados MySQL.

💾

MySQL

Banco de dados responsável pelo armazenamento das informações.
