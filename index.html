Fluent Corp

Aplicativo Mobile para Aprendizado de Idiomas

Projeto acadêmico desenvolvido com Flutter + Node.js + MySQL

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido para auxiliar no aprendizado de idiomas de forma simples e prática. O sistema utiliza um backend em Node.js com Sequelize e banco de dados MySQL, enquanto a interface mobile é construída utilizando Flutter.

Funcionalidades

Cadastro de Usuário

Criação de conta para acessar o sistema.

Login

Autenticação segura de usuários.

Lições

Conteúdos para aprendizado de idiomas.

Progresso

Acompanhamento do desenvolvimento do aluno.

Tecnologias Utilizadas

Flutter
Node.js
Sequelize
MySQL
