Fluent Corp

Aplicativo de aprendizado de idiomas desenvolvido com Flutter e Node.js

Acessar Repositório no GitHub

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas por meio de uma plataforma interativa e acessível.

O sistema utiliza uma arquitetura cliente-servidor composta por um aplicativo mobile desenvolvido em Flutter, uma API em Node.js e banco de dados MySQL.

Tecnologias Utilizadas

Flutter

Framework utilizado para desenvolvimento da interface mobile.

Node.js

Backend responsável pela API e comunicação com o banco de dados.

Sequelize

ORM utilizado para interação com o banco de dados MySQL.

MySQL

Banco de dados responsável pelo armazenamento das informações.

Funcionalidades

  • Cadastro e autenticação de usuários
  • Seleção de idioma e nível de aprendizado
  • Flashcards interativos para estudo
  • Treinamento de conversação
  • Conteúdos em áudio (músicas e podcasts)
  • Perfil do usuário com acompanhamento de progresso
