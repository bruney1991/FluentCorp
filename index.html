Acessar Repositório

Sobre o Projeto

O Fluent Corp é um aplicativo desenvolvido como projeto acadêmico com o objetivo de auxiliar usuários no aprendizado de novos idiomas através de uma interface simples, moderna e interativa.

O sistema utiliza uma API desenvolvida em Node.js com Sequelize para comunicação com banco de dados MySQL e um aplicativo mobile desenvolvido em Flutter.

Tecnologias Utilizadas

📱 Flutter

Framework utilizado para desenvolvimento da interface mobile.

⚙️ Node.js

Responsável pela API e lógica do sistema.

🗄️ MySQL

Banco de dados responsável pelo armazenamento das informações.

🔗 Sequelize

ORM utilizado para comunicação entre Node.js e MySQL.

Arquitetura do Sistema

O sistema segue uma arquitetura cliente-servidor:

  • 📱 Aplicativo Mobile (Flutter)
  • ⚙️ API Backend (Node.js)
  • 🗄️ Banco de Dados (MySQL)

Funcionalidades

Cadastro de usuários
Login de usuários
Seleção de idioma
Definição de nível de conhecimento
Flashcards de aprendizado
Treino de conversação
Conteúdo em áudio (músicas e podcasts)
Planejamento de estudos personalizado
